Project Overview

This project analyzes the sales performance of Blinkit retail outlets using SQL, Python, Excel, and Power BI.
The objective of the project is to clean, analyze, and visualize sales data to identify business insights and trends.

This project demonstrates a complete data analysis workflow including data preprocessing, querying, and dashboard creation.

Tools and Technologies Used
SQL – Data querying and analysis
Python – Data cleaning and preprocessing
Microsoft Excel – Data preparation
Power BI – Data visualization and dashboard creation

Python Libraries Used:

Pandas
NumPy
Matplotlib
Project Workflow
1. Data Collection

The dataset contains information about:

Item Type
Outlet Location
Outlet Size
Outlet Type
Fat Content
Sales
Customer Rating
2. Data Cleaning

Data cleaning was performed using Python and Excel:

Removed missing values
Standardized category names
Cleaned text fields
Prepared the dataset for analysis
3. Data Analysis

SQL queries were used to analyze:

Sales by product category
Sales by outlet location
Sales by outlet type
Average ratings by outlet
4. Dashboard Creation

An interactive dashboard was built using Power BI to visualize:

Total Sales
Average Sales
Number of Items Sold
Average Customer Rating
Sales by Item Category
Sales by Outlet Location
Sales by Outlet Size
Sales Trend by Establishment Year

Filters were implemented for outlet location, outlet size, and outlet type.

Key Insights

Total Sales: $1.20M

Total Items Sold: 8523

Average Sales: $140.99

Average Rating: 3.92

Major Findings:

Tier 3 outlets generated the highest sales.
Supermarket Type 1 outlets sold the highest number of items.
Regular fat products generated more revenue than low-fat products.
Fruits, snacks, and household items were the top-selling categories.
Dashboard Preview

Business Recommendations
Expand operations in Tier 3 cities where sales are highest.
Focus on high-performing product categories such as fruits and snacks.
Improve customer satisfaction to increase average ratings.
